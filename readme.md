# Vid Deepfake Detector

This project provides a Streamlit application for deepfake detection on video files using PyTorch based models. The app supports single video analysis and batch processing of folders.

## Features
- Streamlit interface  
- GPU support when available  
- Single video and folder modes  
- Adjustable frame sampling  
- Choose VAE, ED, or both models  

## Requirements
- Python 3.10  
- PyTorch  
- Streamlit  
- Pandas  

Install dependencies:
```bash
pip install -r requirements.txt


## Usage

Run the app:
streamlit run app.py
